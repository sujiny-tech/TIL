# C언어 연동
> c언어로 만든 소스나 필요한 함수를 go에서 사용하는 법 메모📝   


## 연동을 위한 세팅
+ Ubuntu
   + `sudo apt-get install gcc` 를 통해 gcc 설치

+ Centos
   + `sudo yum install gcc` 를 통해 gcc 설치

+ Windows
   + https://www.mingw-w64.org/ 에서 MinGW-W64 설치
