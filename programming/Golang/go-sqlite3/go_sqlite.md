## 관련해서 발생한 에러들 🖥
+ `UNIQUE constraint failed : User.Id`
   + 이미 저장되어있는 데이터 값이 중복되는 경우 발생 ❗
   + 어떤 컬럼에 UNIQUE 제약조건을 설정하면, 해당하는 컬럼에 중복된 값이 저장될 수 없음.
